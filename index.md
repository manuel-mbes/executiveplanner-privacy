# Privacy Policy

Last updated: 2026

365ExecutivePlanner is a private enterprise calendar planning application developed by MB Engineering Solutions.

## Information We Access

The application connects to Microsoft Outlook calendars using Microsoft authentication.
The app accesses calendar events to display them within the application.

The app may access:

* Calendar event titles
* Event dates and durations

This information is used only to display the calendar within the application.

## Data Storage

Authentication tokens are stored securely on the device using Apple Keychain and secure storage mechanisms.
Calendar information is not stored on external servers.

## Data Sharing

The application does not sell, rent, or share user data with third parties.

## Tracking

The application does not track users and does not use advertising identifiers.Contact
